    Run a node (overview)

    • Hardware: 6+ CPU cores, 16 GB RAM, 100 GB storage, RTX 3060+ recommended.
    • OS: Ubuntu 22.04 LTS for reliability; Windows works but Linux preferred.
    • Drivers: Latest NVIDIA + CUDA; on AMD, check ROCm compatibility.
    • Ops: Monitor logs, keep system updated, allocate stable network bandwidth.
